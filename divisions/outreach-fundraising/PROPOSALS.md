# Community outreach and fundraising division — Proposals

New project ideas and proposals for outreach, community building, and fundraising. Anyone can propose — add yours below and submit a PR.

## How to propose

1. Copy the proposal template below
2. Fill it in at the bottom of this file
3. Submit a PR — discussion happens in the PR comments and on Discord
4. Once accepted (via lazy consensus or council decision), the proposal moves to [Active projects](ACTIVE_PROJECTS.md)

## Proposal template

```markdown
### [Proposal title]

**Proposed by:** @yourhandle
**Date:** YYYY-MM-DD
**Status:** Open / In discussion / Accepted / Withdrawn

**Summary:** One paragraph describing what you want to do.

**Goal:** What outcome are you aiming for? (e.g., X followers, Y grant submitted, Z event organized)

**Approach:** How would you do it? Timeline, channels, partnerships, logistics.

**Resources needed:** People, budget, tools, accounts — what does this require?

**Discussion:** Link to Discord thread or GitHub Discussion if applicable.
```
