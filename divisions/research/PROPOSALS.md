# Research division — Proposals

New project ideas and proposals for research and evaluation. Anyone can propose — add yours below and submit a PR.

## How to propose

1. Copy the proposal template below
2. Fill it in at the bottom of this file
3. Submit a PR — discussion happens in the PR comments and on Discord
4. Once accepted (via lazy consensus or council decision), the proposal moves to [Active projects](ACTIVE_PROJECTS.md)

## Proposal template

```markdown
### [Proposal title]

**Proposed by:** @yourhandle
**Date:** YYYY-MM-DD
**Status:** Open / In discussion / Accepted / Withdrawn

**Summary:** One paragraph describing the research question or evaluation you want to pursue.

**Motivation:** Why is this important? What gap in knowledge does it address?

**Approach:** Methodology, experimental design, data sources, analysis plan.

**Resources needed:** People, compute, hardware, datasets, funding — what does this require?

**Expected outputs:** Papers, datasets, benchmarks, reports — what will this produce?

**Discussion:** Link to Discord thread or GitHub Discussion if applicable.
```
