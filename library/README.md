# Library

This is the shared content space for the Entropic Science community. Anyone can contribute research notes, project documentation, writeups, essays, and more.

## Folders

| Folder | What goes here |
|--------|----------------|
| `research/` | Research notes, experiment reports, literature reviews, data analyses, paper drafts |
| `projects/` | Project documentation, progress reports, technical writeups, architecture decisions |
| `writeups/` | Essays, blog drafts, explainers, opinion pieces, tutorials, guides |

## How to contribute

1. Pick the folder that best fits your content
2. Copy the `_TEMPLATE.md` file in that folder
3. Rename it descriptively — use lowercase, hyphens, and include a date if relevant
   - Good: `qrng-latency-benchmarks-2026-04.md`
   - Good: `consciousness-alignment-literature-review.md`
   - Avoid: `my_notes.md`, `draft1.md`, `untitled.md`
4. Fill in the template fields and write your content
5. Submit a pull request

## Guidelines

- **One file per piece of work.** Don't append to someone else's document — create your own.
- **Use the template.** The frontmatter (title, author, date, status) keeps the library navigable.
- **Link, don't duplicate.** If your work builds on something already in the library, link to it rather than copying sections.
- **Status field matters.** Mark your work as `draft`, `in-progress`, `complete`, or `archived`. This helps readers know what to expect.
- **It's okay to share early.** Drafts and work-in-progress are welcome. Mark them as such and iterate in public.
